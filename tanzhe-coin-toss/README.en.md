# Coin Toss · Hear the Blessing

A zero-dependency browser game inspired by a traditional coin-toss attraction seen at Tanzhe Temple. Aim at five hanging coins, control the throwing power, and try to strike the bell in each center.

![Project cover](docs/cover.png)

## Highlights

- Full-screen mouse aiming and hold-Space-to-charge controls
- Two-click alternative for touch devices and mouse-only play
- Distinct bell, coin-body, and miss feedback
- Original procedural background music and sound effects using Web Audio
- Responsive layout for desktop, tablet, and mobile
- No framework, dependencies, build tools, tracking, or external assets

## Run locally

Download the repository and open `index.html` in a modern browser. Audio starts after the first user interaction because of browser autoplay policies.

## Deploy

Push the repository to the `main` branch, then select **GitHub Actions** under **Settings → Pages**. The included workflow deploys the project automatically.
