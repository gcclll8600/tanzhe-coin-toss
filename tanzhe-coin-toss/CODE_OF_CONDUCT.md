# 社区行为准则

我们希望这里成为一个友善、尊重且适合共同创作的社区。

## 我们鼓励

- 尊重不同经验、背景和观点。
- 对事不对人，给出具体、可执行的反馈。
- 接受合理的纠正，并对自己的影响负责。
- 在涉及传统文化与宗教场所时保持审慎和尊重。

## 不可接受的行为

- 骚扰、侮辱、歧视、威胁或人身攻击。
- 未经许可公开他人的私人信息。
- 垃圾信息、恶意破坏或故意干扰协作。
- 提交侵权内容，或冒充寺院、机构及其官方人员。

维护者可以删除不当内容、关闭讨论，或暂时及永久限制参与者。需要报告社区问题时，请通过仓库维护者提供的私密联系方式或 GitHub 的私密安全报告功能联系。
